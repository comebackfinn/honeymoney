## 📞 문제 해결

### 광고가 표시되지 않음
**원인**: AdSense Client ID 미입력 또는 승인 대기  
**해결**:
1. `.env.local` 파일 확인
2. NEXT_PUBLIC_ADSENSE_CLIENT_ID 값 확인
3. AdSense 승인 상태 확인 (24-48시간 소요)

### 개발 서버 실행 안 됨
**원인**: npm 패키지 미설치  
**해결**:
```bash
npm install
npm run dev
```

### 배포 실패
**원인**: netlify.toml 설정 오류  
**해결**:
1. netlify.toml 파일 확인
2. build command 확인: `npm run build`
3. publish directory 확인: `.next`
